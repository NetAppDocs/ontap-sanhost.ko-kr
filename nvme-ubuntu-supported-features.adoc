[role="lead"]
ONTAP 시스템 설정에서 실행되는 ONTAP 버전에 관계없이 다음과 같은 SAN 호스트 기능을 지원합니다.

[cols="75,25"]
|===
| 특징 | Ubuntu 호스트 버전 


| 네이티브 패키지의 NetApp 플러그인은 NVMe/FC 네임스페이스에 대한 ONTAP 세부 정보를 표시합니다.  a| 
22.04 이상



| NVMe 및 SCSI 트래픽은 NVMe 네임스페이스 점유를 방지하기 위한 명시적인 dm-multipath 설정 없이 지정된 호스트 버스 어댑터(HBA)의 동일한 호스트에서 지원됩니다  a| 
22.04 이상



| 커널 내 NVMe 다중 경로는 NVMe 네임스페이스에 대해 기본적으로 활성화되어 있습니다. 따라서 명시적 구성이 필요하지 않습니다.  a| 
22.04 이상

|===
